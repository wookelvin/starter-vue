# Get started

This is a starter frontend boilerplate project for a vue project. 

Using
- vite
- vue3
- typescript
- pinia
- tailwindcss
- postcss with nesting, autoprefixer, importing
- eslintrc
- @ alias setup
- @vueuse/core
- lodash
- iconoir

# To get started 

```bash
npm i -g @antfu/ni
ni
nr dev
```
